# Green-tailwind
The plant store website Greenden is built with HTML5 and Tailwind CSS as a modern responsive design. The project demonstrates various indoor and outdoor plant species through its simple and efficient user-friendly design.

# Project Overview

* Greenden operates as an online plant store which allows users to. 

* Users can check indoor and outdoor plants which the website offers. 

* The plant collection allows users to see plants through visual content. 

* The website provides access to its main contact method through a contact form.

The website uses Tailwind CSS utility classes to create an interface which works properly on different device types.

# Features

* Responsive design (Mobile, Tablet, Desktop)

* Indoor & Outdoor plant sections

* Featured plant collections

* Contact form

  * HTML5

  * Flexbox and Grid
 
  * Responsive Web Design
